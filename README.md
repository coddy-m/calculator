# Calculator

A modern, feature-rich calculator web application with **Unit Converter**, multiple beautiful themes, scientific functions, and persistent history.

Built with pure **HTML, CSS, and JavaScript** — no frameworks required.


## ✨ Features

### Calculator
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific mode with trigonometric functions (sin, cos, tan), logarithms, square root, power, and π
- Memory functions: **MC**, **MR**, **M+**, **M-**
- Percentage and sign toggle (±)
- Backspace and clear functionality
- Full keyboard support (numbers, operators, Enter, Backspace, Escape)

### Unit Converter
- **7 Categories**: Length, Weight, Temperature, Volume, Area, Speed, Time
- Real-time bidirectional conversion
- Supports common units (meters, kg, Celsius/Fahrenheit, liters, etc.)

### History
- Automatically saves all calculations
- Shows expression and result with timestamps
- Persistent storage using `localStorage` (remains after closing browser)
- Clear history option

### UI & Customization
- **6 Beautiful Themes**: Light, Dark, Neon Glow, Pastel Dream, Ocean, Forest
- Smooth tab navigation: Calculator | Converter | History
- Fully responsive and mobile-friendly design
- Modern glassmorphic-style interface with smooth animations

## 🚀 How to Use

1. Download the file `ultimate-calculator.html`
2. Open the file in any modern web browser (Chrome, Firefox, Edge, Safari)
3. No installation or server required — works completely offline

### Switching Modes
- Click **Scientific** / **Basic** to toggle scientific functions
- Use the theme dropdown in the header to change appearance
- Switch between **Calculator**, **Converter**, and **History** using the tabs

## 📱 Keyboard Shortcuts

| Key              | Action                  |
|------------------|-------------------------|
| `0-9`, `.`       | Enter numbers           |
| `+`, `-`, `*`, `/` | Operators             |
| `Enter` or `=`   | Calculate result        |
| `Backspace`      | Delete last character   |
| `Escape`         | Clear display           |

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Transitions)
- Vanilla JavaScript (ES6+)
- `localStorage` for data persistence
